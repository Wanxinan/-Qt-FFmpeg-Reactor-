`C++11` `Qt` `libvlc` `FFmpeg` `MySQL` `Reactor模型` `TLV协议` `MD5/crypt加密` 

- **项目描述**：
    项目面向智能工厂场景，基于 C++/Qt 跨平台开发，构建集用户管理、摄像头监控、视频录制与智能设备信息采集于一体的综合系统。系统支持 RTSP 流实时播放、账户加密认证与日志审计，客户端采用 FFmpeg 实现跨平台流媒体接入与播放功能。
- **工作内容**：
  - 搭建基于 Reactor + Thread的高并发服务器，处理用户注册、登录与设备信息上传任务。
  - 使用 TLV 协议实现服务端通信格式标准化，支持多种类型数据的高效解析与响应。
  - 基于 FFmpeg 框架实现摄像头 RTSP 流的抓取与播放，支持多分辨率和多源接入切换。
  - 使用 Qt 封装 FFmpeg 播放控件，设计本地视频播放窗口与网络流实时渲染逻辑。
  - 完善注册登录机制，集成 MD5 加密与盐值生成算法，保障用户认证数据安全性。
